# Development Environments

## Version 
Middleware required for development and supported versions
```
node >= v22.x
```

## Initial Settings
```
mkdir next2d
cd next2d
git clone git@github.com:Next2D/player.git
git clone git@github.com:Next2D/framework.git
cd framework
npm install
```

## Unit Test
```
npm test
```

## ESLint
```
npm run lint
```
